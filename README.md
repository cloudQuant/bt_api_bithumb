# bt_api_bithumb

Bithumb exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_bithumb
```

## Usage

```python
from bt_api_bithumb import BithumbSpotExchangeData, BithumbFeed
```

## Features

- REST API support
- HMAC-SHA256 authentication
- WebSocket support
- Order book, ticker, kline, balance, account, and order data

## Symbol Format

Bithumb uses hyphen-separated symbols (e.g., `BTC-USDT`).
